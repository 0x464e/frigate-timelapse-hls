# Frigate Timelapse HLS
## Live HLS timelapse stream generation of the current day's recordings

This is pretty much a personal project, but nothing stops you from using it if you want.  
What is the purpose of this? Well imagine you want to review a timelapse of what happened on your camera today, 
sure you can export a timelapse from Frigate at the end of the day. But say it's 3pm and you want to see the
timelapse of the day so far. Well with this project you can, a HLS stream of the current day's timelapse is 
generated. Every time you open the stream with your browser, you can see the current day's timelapse up until this point.
